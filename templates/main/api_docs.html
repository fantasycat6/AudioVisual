📚 API 基础介绍

了解视频解析平台API的基本概念和使用要求

核心功能

视频解析平台提供多个API接口，主要用于：

  • 智能视频解析：自动尝试多个解析接口，直到找到可用的
  • 快速接口获取：获取当前所有可用的解析接口列表
  • 影视导航：获取影视资源导航网站列表
  • 用户管理：用户认证和权限验证

认证要求

所有API接口都需要用户登录认证，通过以下方式验证：

  • 使用 Flask-Login 进行用户会话管理
  • 每个请求都需要有效的登录会话Cookie
  • 部分接口需要管理员权限（如管理后台相关API）

通用响应格式

所有JSON API接口都遵循统一的响应格式：

{
    "success": true,
    "message": "操作成功说明",
    "data": { /* 具体的返回数据 */ }
}

或失败时：

{
    "success": false,
    "message": "错误信息说明",
    "error": "具体的错误类型"
}

💻 接口使用示例

实际调用API接口的代码示例

JavaScript (Fetch API)

// 智能解析API调用示例
async function smartParseVideo(videoUrl) {
    try {
        const response = await fetch(`/api/parse/smart?video_url=${encodeURIComponent(videoUrl)}`);
        const data = await response.json();
        
        if (data.success) {
            console.log('解析成功:', data.parse_url);
            // 加载视频播放器
            document.getElementById('video-iframe').src = data.parse_url;
        } else {
            console.error('解析失败:', data.message);
        }
    } catch (error) {
        console.error('请求失败:', error);
    }
}

// 获取解析接口列表
async function getParseAPIs() {
    try {
        const response = await fetch('/api/parse/quick');
        const data = await response.json();
        
        if (data.success) {
            console.log('可用接口:', data.apis);
            return data.apis;
        }
    } catch (error) {
        console.error('获取接口失败:', error);
    }
}

Python (requests库)

import requests

def smart_parse_video(video_url):
    """智能解析视频链接"""
    url = f"http://your-domain/api/parse/smart"
    params = {"video_url": video_url}
    
    # 需要先登录获取session
    session = requests.Session()
    # 登录代码略...
    
    response = session.get(url, params=params)
    data = response.json()
    
    if data['success']:
        print(f"解析成功，使用接口: {data['api_name']}")
        print(f"播放链接: {data['parse_url']}")
        return data['parse_url']
    else:
        print(f"解析失败: {data['message']}")
        return None
