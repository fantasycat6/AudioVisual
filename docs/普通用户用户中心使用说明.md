# 普通用户用户中心使用说明

## 功能概述
为解决普通用户无法修改密码的问题，我们新增了完整的用户中心功能，让所有用户都能自助管理账户信息。

## 功能特性

### 1. **用户中心入口**
- **位置**：页面右上角导航栏
- **入口1**：点击用户名区域（显示用户图标和用户名）
- **入口2**："用户中心"按钮
- **效果**：所有登录用户（无论管理员还是普通用户）都能访问自己的用户中心

### 2. **用户中心页面结构**

#### **个人信息区域**
- **用户图标**：显示管理员👑或普通用户👤图标
- **用户名**：当前登录用户的用户名
- **邮箱**：用户注册时使用的邮箱
- **身份标识**：显示是管理员还是普通用户（不同颜色的徽章）

#### **账户统计信息**
- **注册时间**：用户的注册日期和时间
- **最后登录**：最近一次成功登录的时间
- **自动更新**：修改密码或登录时会更新最后登录时间

#### **功能卡片**
1. **🔒 修改密码**
   - 用户可自助修改自己的密码
   - 需要验证当前密码
   - 新密码需满足安全要求

2. **👑 申请管理员权限**（仅普通用户可见）
   - 显示说明：需要联系现有管理员授权
   - 按钮处于禁用状态（无法直接申请）

3. **⚙️ 管理后台**（仅管理员可见）
   - 管理员专用入口
   - 点击直接进入系统管理后台

4. **🚨 账户安全**
   - 安全建议和最佳实践
   - 密码管理指导
   - 账户保护措施

### 3. **密码修改流程**
```
点击"修改密码" → 弹出验证窗口 → 输入：
1. 当前密码（验证身份）
2. 新密码（至少6个字符）
3. 确认新密码（两次输入一致）

→ 验证通过 → 密码修改成功 → 自动更新最后登录时间
```

### 4. **安全提示功能**
- 提供5项重要的账户安全建议
- 包含密码要求、更换频率、共享风险等内容
- 清晰的图标和文字说明

## 错误处理

### 常见错误及解决
| 错误信息 | 原因 | 解决方案 |
|---------|------|---------|
| "旧密码错误" | 当前密码输入不正确 | 检查密码是否正确，注意大小写 |
| "新密码至少需要6个字符" | 密码太短 | 使用至少6个字符的密码 |
| "新密码和确认密码不匹配" | 两次输入不一致 | 确保完全相同的密码 |
| "旧密码和新密码不能为空" | 漏填字段 | 填写所有必要字段 |

### 网络错误处理
- 显示友好的错误消息
- 按钮禁用防止重复提交
- 表单自动重置，避免隐私泄露

## 安全措施

### 1. **数据安全**
- 所有密码使用SHA-256哈希存储
- 前端验证后后端再次验证
- 会话管理和CSRF保护

### 2. **操作安全**
- 必须验证当前密码才能修改
- 修改成功后自动退出，需要重新登录
- 记录最后登录时间，帮助识别异常

### 3. **隐私保护**
- 只显示自己的信息，无法查看他人信息
- 邮箱等敏感信息妥善显示
- 密码输入框使用password类型

## 故障排除指南

### 问题1：无法访问用户中心
**可能原因**：
1. 未登录（需要先登录）
2. 浏览器缓存问题
3. JavaScript错误

**解决方案**：
1. 确认已登录，右上角显示用户名
2. 刷新页面或清除浏览器缓存
3. 检查浏览器控制台错误信息

### 问题2：密码修改不生效
**可能原因**：
1. 网络连接问题
2. 服务器内部错误
3. 数据库事务失败

**解决方案**：
1. 检查网络连接
2. 查看服务器日志
3. 尝试重新操作

### 问题3：样式显示异常
**可能原因**：
1. CSS文件加载失败
2. 浏览器兼容性问题
3. 缓存CSS文件过时

**解决方案**：
1. 检查开发者工具Network面板
2. 使用标准现代浏览器
3. 强制刷新页面（Ctrl+F5）
